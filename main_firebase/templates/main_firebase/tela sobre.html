Este é um projeto desenvolvido utilizando djago 4.2 com as bibiliotecas presentes na tabela 1: bibliotecas do projeto.

Este projeto usa um serviço de autenticação do firebase chamado Firebase Authenticator para validar usuarios, para isso é usada a biblioteca pip pyrebase.

Para manter os dados persistentes no banco de dados é usado o SGBD Mysql conectado segundo o arquivo settings.py

Este aplicativo cadaStra usuarios reseta a conta em caso de "esqueci a senha" exibe interface de admin para usuarios authenticados e fornece uma listagem de conteudo do banco para rodar o aplicativo connectado ao database com os dados persistentes.
